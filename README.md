# 🗂️ Plantilla HECSAB

Plantilla profesional para proyectos de **Análisis de Datos** creada por Sabín Areiza
con la colaboración de Hector Lavoe.

Inspirada en el modelo Cookiecutter Data Science y diseñada para tener una estructura
ordenada, clara y lista para trabajar desde el primer momento.

---

## 🚀 ¿Cómo usar esta plantilla?

Asegúrate de tener instalado uv Astral en tu computador.
Si no lo tienes, instálalo desde: https://astral.sh/uv

Luego ejecuta este comando en tu terminal:

uvx cookiecutter gh:sareiza/plantilla_hecsab

Responde las preguntas que aparecen y listo. Tu proyecto estará completamente configurado.

---

## 📁 Estructura del proyecto

Cada proyecto creado con esta plantilla tendrá la siguiente estructura:
```
Estructura del Carpetado  

mi_proyecto/
├── data/
│   ├── raw/           → Datos originales sin modificar
│   ├── processed/     → Datos limpios y transformados
│   └── external/      → Datos de fuentes externas
├── notebooks/         → Jupyter Notebooks para exploración y experimentación de datos
├── diagrams/          → Diagramas y modelado de datos en formato PNG
├── docs/              → Documentación en Markdown: análisis realizados con su
│                        respectivo razonamiento y conclusiones, hallazgos con su
│                        profundización, identificación de patrones y demás análisis
│                        relevantes para el desarrollo del proyecto
├── grafics/           → Gráficos generados con matplotlib y seaborn
├── src/
│   ├── services/      → Lógica principal del proyecto
│   └── routes/        → Puntos de entrada de la aplicación
├── scripts/           → Scripts de Python con el código modularizado
├── .gitignore         → Archivos y carpetas ignorados por Git
├── main.py            → Punto de entrada principal del proyecto
└── README.md          → Documentación general del proyecto
```
